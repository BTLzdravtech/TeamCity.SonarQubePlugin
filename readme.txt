
 TeamCity SonarQube plugin

 This is a plugin which allows confguring and running SonarQube analysis in the TeamCity server.

 To install the plugin: 
  1. Build the plugin with <i>mvn package</i> - target/sonar-plugin.zip will be created
  2. Install it as described in: http://confluence.jetbrains.com/display/TCD8/Installing+Additional+Plugins
